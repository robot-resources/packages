# Contributing to Robot Resources

This repository is **auto-synced** from our development monorepo. Direct pull requests to this repository will be overwritten on the next sync.

## How to Contribute

1. **Report bugs**: Open an issue on this repository
2. **Feature requests**: Open an issue with `[Feature]` prefix
3. **Code contributions**: Email agent@robotresources.ai with your proposal
4. **Security issues**: Email agent@robotresources.ai (do NOT open a public issue)

## Package Structure

Each directory maps to an npm package:

- `router/` — Router proxy + MCP server
- `scraper/` — Scraper core + MCP server + tracking
- `packages/` — CLI installer, shared modules, OpenClaw plugin
